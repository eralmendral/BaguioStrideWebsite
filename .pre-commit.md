# Pre-commit Checklist

Before committing your changes, make sure to:

1. **Run BiomeJS checks:**
   ```bash
   npm run check:fix
   ```

2. **Build the project to ensure everything compiles:**
   ```bash
   npm run build
   ```

3. **Test locally:**
   ```bash
   npm run preview
   ```

4. **Verify .npmrc is not committed** (it contains your HugeIcons license key)

5. **Ensure all environment variables are in .gitignore**
